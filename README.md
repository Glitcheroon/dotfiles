A simple and minimalist rice for i3 WM meant for Arch Linux

Packages:
 - picom
 - polybar
 - cava
 - fastfetch
 - rofi
 - ibus (ibus-bamboo)
 - maim
 - feh


Keybinds:
 - Super + D: Rofi search
 - Super + Enter: Open terminal
 - PrtSc: maim fullscreen screenshot
 - Super + F5: Select audio output
 - Super + F4: Wallpaper selector
 - Super + F1: Open blueman-manager
 - Super + Space: language switch (VN / EN)
